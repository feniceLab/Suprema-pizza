# Índice — Regras de Skill (Suprema Pizza)

## O que é essa pasta

Esta pasta contém as **regras consultadas pela skill `stop-motion-creator`** (e futuras skills da Fenice Lab) antes de gerar qualquer conteúdo para a Suprema Pizza.

O objetivo é garantir **padronização total da identidade visual e do tom de voz** em todas as peças produzidas — independentemente de qual IA, agente ou colaborador esteja executando a tarefa.

Sempre que houver mudança aprovada com o cliente, atualizar o arquivo correspondente e registrar a data na seção de versionamento de cada documento.

## Arquivos desta pasta

| Arquivo | Conteúdo |
|---------|----------|
| `00 - Indice.md` | Este arquivo — visão geral e mapa da pasta |
| `01 - Identidade Visual.md` | Cores, fontes, halo de legenda, especificações do logo |
| `02 - Tom de Copy.md` | Padrão de copy em 2 linhas (gancho + revelação), vibe, exemplos aprovados |
| `03 - Padroes Tecnicos.md` | Specs FFmpeg/ImageMagick: resolução, codecs, animações, mapeamento de música |

| `05 - Como a Skill Consulta.md` | Instruções para a IA sobre como ler e aplicar essas regras |
